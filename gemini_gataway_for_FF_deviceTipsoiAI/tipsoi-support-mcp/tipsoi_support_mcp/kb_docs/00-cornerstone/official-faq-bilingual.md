## FAQ 01: আজকে সিস্টেমে ডাটা দেখা যাচ্ছে না / Attendance আসেনি
## FAQ 01: No Data Showing Today / Attendance Not Coming

### ক্লায়েন্ট সাধারণত কী প্রশ্ন করে? / What clients ask:
**বাংলা:**
- আজকে সিস্টেমে কোনো ডাটা দেখা যাচ্ছে না কেন?
- Attendance punch হয়েছে, কিন্তু dashboard এ show করছে না কেন?
- Device punch নিচ্ছে, কিন্তু software এ data আসছে না কেন?

**English:**
- Why is no data showing in the system today?
- Attendance punch happened but the dashboard is not showing it
- Device is punching but data is not coming to the software

---

### সমস্যা চেনার উপায় / How to identify:

**যদি Device communicate না করে:** তাহলে বুঝি network / SIM Balance / WiFi problem।

**If device is not communicating:** This means network / SIM Balance / WiFi problem.

---

### সমস্যার সমাধান / Solution:

**Step 1: Device Communication Check**
- **বাংলা:** ক্লায়েন্টকে বলা হয়, Tipsoi Admin Panel-এ login করে sidebar থেকে **Devices** option-এ click করতে। এরপর সংশ্লিষ্ট device-এর **Last Communication Time** দেখে আমাদের সাথে share করতে।
- **English:** Ask client to log into Tipsoi Admin Panel and click **Devices** from the sidebar. Check the device's **Last Communication Time**.

**Step 2: TF 80 Fingerprint Device (SIM Based) হলে / If TF 80:**
- **বাংলা:** Device এর SIM number check করা হয়। SIM active আছে কিনা দেখা হয়। Recharge / due date শেষ হয়েছে কিনা check করা হয়। Recharge করলে সাধারণত device আবার communicate শুরু করে।
- **English:** Check the SIM number in device. Check if SIM is active. Check if recharge/due date has passed. After recharging, the device usually starts communicating again.

**Step 3: Fast Face Device (WiFi) হলে / If Fast Face:**
- **বাংলা:** Client কে WiFi reconnect করতে বলা হয়। Process: Password → 1 থেকে 6 → Settings → Network → WiFi → নিজের WiFi password দিয়ে connect
- **English:** Ask client to reconnect to WiFi. Process: Password (1-6) → Settings → Network → WiFi → connect using office WiFi password

**Step 4: Prompt Series Device (WiFi) হলে / If Prompt Series:**
- **বাংলা:** WiFi reconnect করার জন্য client কে ভিডিও guideline provide করা হয়।
- **English:** Provide client with a video guideline to reconnect WiFi.

**Step 5: সমাধান হয়েছে কিনা চেক / Confirm resolution:**
- **বাংলা:** Device communicate শুরু করলে client কে confirm করা হয়। এরপর dashboard/report-এ data update হয়েছে কিনা check করতে বলা হয়।
- **English:** When device communicates, confirm with client. Then check if data appears in dashboard/reports.

## FAQ 02: লগইন করতে পারছি না / Login Issue
## FAQ 02: Cannot Log In

### ক্লায়েন্ট সাধারণত কী প্রশ্ন করে? / What clients ask:
**বাংলা:**
- আমি login করতে পারছি না কেন?
- Email ও password ঠিক দিয়েও login হচ্ছে না
- বারবার incorrect Credential দেখাচ্ছে
- App/Web login নিচ্ছে না কেন?

**English:**
- Why can I not log in?
- Email and password are correct but login is not working
- "Incorrect Credentials" keeps showing
- App/Web login is not working

**Login URL:** https://hrm.tipsoi.pro/login

---

### সমস্যা চেনার উপায় / How to identify:
**বাংলা:**
1. যদি আমরা same ID দিয়ে login করতে পারি → তাহলে বুঝি Typing Mistake
2. যদি আমরাও login না পারি → তাহলে বুঝি credential wrong
3. যদি অনেক user একই complain করে → তাহলে বুঝি server down issue

**English:**
1. If support can log in with same credentials → Typing mistake by client
2. If support also cannot log in → Credentials are wrong
3. If many users complain simultaneously → Server down issue

---

### সমস্যার সমাধান / Solution:

**Step 1: Email & Password Verify**
- **বাংলা:** প্রথমে client কে email এবং password আবার সঠিকভাবে দিতে বলা হয়। অনেক সময় typing mistake এর কারণে login হয় না।
- **English:** Ask client to carefully re-enter email and password. Many times login fails due to typing mistakes.

**Step 2: Credentials Collect & Check**
- **বাংলা:** যদি তবুও login না হয়, তাহলে client কে WhatsApp এ email ও password পাঠাতে বলা হয়। এরপর support team system-এ check করে login test করে দেখে।
- **English:** If still failing, ask client to send credentials via WhatsApp. Support team tests login from their side.

**Step 3: যদি Login হয়ে যায় / If Login Works:**
- **বাংলা:** Client কে জানানো হয় credentials ঠিক আছে। সম্ভবত network, browser, app cache বা device issue এর কারণে login হচ্ছিল না।
- **English:** Inform client credentials are correct. Issue was likely network, browser cache, or device problem.

**Step 4: যদি System Down থাকে / If System Down:**
- **বাংলা:** Client কে জানানো হয় system-এ সাময়িক সমস্যা চলছে। Technical team বিষয়টি নিয়ে কাজ করছে। সমস্যা সমাধান হলে update দেওয়া হবে।
- **English:** Inform client there is a temporary system issue. Technical team is working on it. Update will be provided when resolved.

## FAQ 03: Location Tracking দেখা যাচ্ছে না / লোকেশন রিপোর্ট আসছে না
## FAQ 03: Location Tracking Not Showing

### ক্লায়েন্ট সাধারণত কী প্রশ্ন করে? / What clients ask:
**বাংলা:**
- Employee-এর location দেখা যাচ্ছে না কেন?
- Location tracking report blank দেখাচ্ছে কেন?
- location update আসছে না কেন?
- লোকেশন tracking on আছে, তবুও data আসছে না কেন?

**English:**
- Why is employee location not visible?
- Location tracking report is blank
- Location update is not coming
- Tracking is on but data is not showing

---

### সমস্যা চেনার উপায় / How to identify:
**বাংলা:**
- যদি শুধু একজন Employee-এর location data না আসে → সাধারণত বুঝা যায় এটি ওই Employee এর mobile setting issue (internet off, GPS/location off, app permission বন্ধ, battery restriction)
- যদি একাধিক বা সকল Employee-এর data না আসে → সাধারণত বুঝা যায় এটি Admin setting issue অথবা system side issue (admin panel থেকে Location tracking setting inactive)

**English:**
- If only one employee's location is missing → That employee's mobile setting issue (internet off, GPS/location off, app permission blocked, battery restriction)
- If all employees' locations are missing → Admin setting issue or system issue (location tracking setting inactive in admin panel)

---

### সমস্যার সমাধান / Solution:

**Step 1: Admin Settings Check**
- **বাংলা:** প্রথমে check করা হয় employee এর জন্য location tracking option active আছে কিনা।
- **English:** First check if location tracking option is active for the employee in admin settings.

**Step 2: Employee Mobile Settings Check**
- **বাংলা:** Employee এর মোবাইলে নিচের বিষয়গুলো active আছে কিনা নিশ্চিত করা হয়: Mobile Data On, GPS / Location On, App Background Running
- **English:** Confirm these are active on employee's phone: Mobile Data On, GPS/Location On, App Background Running

**Step 3: Video Call Support**
- **বাংলা:** যদি সব ঠিক থাকার পরও location না আসে, তাহলে client এর WhatsApp number নেওয়া হয়। Video call / screen share এর মাধ্যমে settings check করা হয়।
- **English:** If still not working, get client's WhatsApp number. Check settings via video call/screen share.

**Step 4: Permission & App Check — Screen share এ দেখা হয়:**
- **বাংলা:** Location permission allowed আছে কিনা, Mobile data on আছে কিনা, GPS চালু আছে কিনা, App background-এ আছে কিনা, App latest version update আছে কিনা
- **English:** Check: Location permission Allowed, Mobile data On, GPS On, App running in background, App updated to latest version

**Step 5: Refresh & Recheck**
- **বাংলা:** Client কে আবার app open করে বের হতে বলা হয়। এরপর report reload করে location data এসেছে কিনা check করা হয়।
- **English:** Ask client to close and reopen app. Then reload Location Tracking Report.

### সঠিক Location Tracking-এর জন্য / For accurate location tracking:
- Mobile Data সবসময় on রাখতে হবে / Keep Mobile Data always ON
- GPS / Location on রাখতে হবে / Keep GPS/Location ON
- Battery Optimization off রাখতে হবে / Turn Battery Optimization OFF
- App background-এ running রাখতে হবে / Keep app running in background
- App update রাখা ভালো / Keep app updated

## FAQ 04: Face Photo Upload Not Working

### ক্লায়েন্ট সাধারণত কী প্রশ্ন করে? / What clients ask:
**বাংলা:**
- আমি ছবি allocate করছি কিন্তু নিচ্ছে না কেন?
- Photo upload দিচ্ছি, save হচ্ছে না কেন?

**English:**
- I am allocating a photo but it is not being accepted
- Photo upload is not saving
- Face image is given but not detecting
- Cannot add photo for an employee

---

### সমস্যা চেনার উপায় / How to identify:
**বাংলা:**
- যদি image-এ face clear না থাকে → বুঝি এটি image quality issue (blur ছবি, কম আলো, low resolution, face স্পষ্টভাবে দেখা যাচ্ছে না)
- যদি background-এ অনেক object থাকে → বুঝি device face detect করতে সমস্যা করছে

**English:**

**Step 1: Photo Quality Check**
- **বাংলা:** প্রথমে client-কে জিজ্ঞেস করা হয় ছবিতে face clear আছে কিনা। Background solid color কিনা check করা হয়।
- **English:** Ask client if face is clear in the photo. Check if background is solid color.

**Step 2: WhatsApp-এ Photo Review**
- **বাংলা:** যদি client নিশ্চিত না হন, তাহলে photo WhatsApp-এ পাঠাতে বলা হয়। Support team photo দেখে requirement অনুযায়ী guide করে।
- **English:** If client is unsure, ask them to send photo via WhatsApp. Support team reviews and guides.

**Step 3: সঠিক Photo Requirement / Correct Photo Requirements:**
**বাংলা:** ভালো allocation-এর জন্য ছবিটি হতে হবে:
- Passport size style photo
- Face clear ও সামনে থেকে visible
- ভালো lighting থাকা
- Background solid/plain color
- Blur না হওয়া

**English:** For good face allocation, photo must be:
- Passport size style (face close-up)
- Face clearly visible, facing forward
- Good lighting
- Solid/plain background
- Not blurry

**Step 4: Photo Crop & Retry**
- **বাংলা:** যদি photo অনেক বড় হয় বা full image হয়, তাহলে face focus করে crop করতে বলা হয়। এরপর আবার upload/allocate করতে বলা হয়।
- **English:** If photo is too large or full-body, crop focusing on face only. Then retry upload/allocation.

**Step 5: Final Allocation Support**
- **বাংলা:** প্রয়োজনে support team crop করা photo দিয়ে allocate করতে সহায়তা করে।

## FAQ 05: Punch করেছে কিন্তু In Time / Out Time আসেনি
## FAQ 05: Punched but In Time / Out Time Not Showing

### ক্লায়েন্ট সাধারণত কী প্রশ্ন করে? / What clients ask:
**বাংলা:**
- Employee in time punch করেছে, কিন্তু In Time আসেনি কেন?
- Out time punch করেছে, কিন্তু Out Time show করছে না কেন?
- শুধু In Time আসছে, Out Time আসছে না কেন?
- Punch log আছে, কিন্তু attendance summary-তে দেখাচ্ছে না কেন?

**English:**
- Employee punched in time but In Time is not showing
- Out time punch happened but Out Time is not showing
- Only In Time shows, Out Time does not show
- Punch log has entry but attendance summary is not showing it

---

### মূল ধারণা: Parenting Time / Core Concept: Parenting Time
**বাংলা:**
- Punch যদি valid parenting time / allowed range-এর মধ্যে থাকে → In Time / Out Time হিসেবে show করবে
- Punch যদি allowed range-এর বাইরে থাকে → শুধু log হিসেবে থাকবে, summary attendance-এ show করবে না

System shift start/end time অনুযায়ী নির্দিষ্ট allowance time এর মধ্যে punch count করে। সাধারণত shift time থেকে নির্দিষ্ট limit (যেমন ৬ ঘন্টা) এর বাইরে punch করলে সেটি attendance summary-তে In/Out Time হিসেব আসে না।

**English:**
- If punch is within the valid Parenting Time / allowed range → it shows as In Time / Out Time
- If punch is outside the allowed range → it only appears as a log entry, NOT in attendance summary

The system counts punches only within a defined time window around the shift start/end time. Punches outside this window (typically 6 hours from shift time) do not appear as In/Out Time in the attendance summary.

---

### সমস্যার সমাধান / Solution:

**Step 1: Punch Confirm করা**
- **বাংলা:** প্রথমে client-এর কাছ থেকে confirm করা হয় employee আসলে punch করেছে কিনা।
- **English:** First confirm from client that the employee actually punched.

**Step 2: Shift Time নেওয়া**
- **বাংলা:** Employee-এর shift time জেনে নেওয়া হয়।
- **English:** Find out the employee's shift time.

**Step 3: Punch Time Check করা**
- **বাংলা:** Employee কখন punch করেছে তা জিজ্ঞেস করা হয়।
- **English:** Ask what time the employee punched.

**Step 4: Parenting Time Check করা**
- **বাংলা:** Punch time যদি shift time থেকে অনেক দূরে হয় (allowed range-এর বাইরে) → attendance summary-তে আসবে না। শুধু Detail Attendance Log-এ থাকবে।
- **English:** If punch time is far from shift time (outside allowed range) → it will not appear in attendance summary. Only visible in Detail Attendance Log.

**Step 5: Detail Attendance Log Check**
- **বাংলা:** Employee profile-এর Detail Attendance Log থেকে দেখা হয়: কতবার punch করেছে, কখন punch করেছে
- **English:** Check from employee profile's Detail Attendance Log: how many times punched, when punched

To access: **Employee Profile → Detail Attendance Log**
