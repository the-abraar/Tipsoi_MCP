# FAQ - Attendance, Login & Location Tracking Issues

## FAQ 01: Attendance Issues (আপনাকে মিস্টেমে উটা দেখা যাচ্ছে না)

### What should I do if attendance is not showing?

Common reasons and solutions:

**1. Device Communication Issue**
- Check if the device can communicate with the system
- Verify device is connected to internet/network
- Check for network connectivity and WiFi problems
- Verify device SIM balance (for SIM-based devices)

**2. Dashboard Not Updating**
- Attendance punch may not reflect immediately
- Check if dashboard shows the punch record after refresh
- Device may take time to sync data with server

**3. Software/Device Data Mismatch**
- Device punch exists but not showing in software
- Software data exists but dashboard doesn't show it
- Data synchronization delay between device and system

### How to Troubleshoot Attendance Issues?

#### Step 1: Device Communication Check
- Log into Tipsoi Admin Panel
- Navigate to: Sidebar → Devices option
- Check "Last Communication Time" for each device
- Verify device details match the setup

#### Step 2: For TF 80 Fingerprint Device (SIM Based)
- Check the device's SIM number
- Verify SIM is active with sufficient balance
- Check recharge status and due date
- Recharge the device SIM after charging to establish communication

#### Step 3: For Fast Face Device
- Ask the client to WiFi reconnect
- Navigate through: Settings → Network → WiFi → Connect WiFi password

#### Step 4: For Prompt Series Device
- Provide client with WiFi connection guidelines
- Reconnect the client to the correct WiFi network

### How to Troubleshoot If Device Won't Communicate?

**Customer's Troubleshooting Steps:**
- Check if the customer can confirm device connection via the interface
- Update the dashboard and report data
- Check if reported punch data appears in customer's reports

**If Communication Still Fails:**
- Customer should contact check point or request confirmation from customer
- If customer confirms after troubleshooting, escalate to support
- Note: Customer may need a quick trick to resolve the issue
- If still failing, follow the standard support escalation process

## FAQ 02: Login Issues (লগইন করতে পারছি না)

### What should I do if I can't log in?

Common login issues and their solutions:

**1. Email/Password Incorrect**
- Make sure you're using the correct email or Employee ID
- Verify that the correct password is being entered
- Check for incorrect credentials being entered

**2. Wrong Credentials**
- Email or password might be typed incorrectly
- If you suspect someone else has your account, check if another user is logged in
- Contact support if you're locked out

**3. App or Web Login Issues**
- Can't log in from mobile app
- Can't log in from web browser
- Unable to login on either platform

### Step-by-Step Login Troubleshooting

#### Step 1: Email & Password Verification
- Ask client to verify email and password are correct
- Eliminate any typing mistakes in login attempt
- Try logging in again with correct credentials

#### Step 2: Credentials Collection & Check
- If login still fails, collect credentials from the client via WhatsApp
- Ask support team to check if login works with those credentials
- Perform login test in the support system

#### Step 3: Check for Login Issues
- Client may not be in the system
- Credentials might be incorrect or server issue
- Network, browser, app cache, or device issue

#### Step 4: System Down?
- If multiple clients report login failures, check if system is down
- Contact technical team to investigate
- System may be under update or experiencing issues

### Troubleshooting Login If Problem Persists

**Customer's Troubleshooting Steps:**
- Verify email and password are correct
- Check internet connectivity and signal strength
- Clear browser/app cache and try again
- Try logging in from different device or network
- Check if this is a temporary network or device issue

**If Customer Still Can't Login:**
- Contact support for account verification
- Support will reset password or check account status
- Technical team can investigate system-level issues

**Final Support Response**
- Support escalates to technical team if needed
- Resolve within agreed SLA
- Provide regular updates to customer on issue status

## FAQ 03: Location Tracking Issues (দেখা যাচ্ছে না / লোকেশন ট্র্যাকিং আসছে না)

### What should I do if location is not showing?

Common location tracking issues:

**1. Employee's Location Data Not Showing**
- Mobile data might be off or insufficient
- GPS/Location services disabled on phone
- Background app restrictions blocking location access

**2. Location Tracking Report Blank**
- No location update showing in reports
- Location data is missing from the system

**3. Location Update Not Coming**
- Real-time location tracking not active
- Data not syncing to the dashboard

### Key Troubleshooting Steps

#### Step 1: Admin Settings Check
- First, verify location tracking is active for the employee
- Navigate to: Admin Panel Settings
- Check if location tracking option is enabled in organization settings

#### Step 2: Employee Mobile Settings Check
- Verify the following settings are enabled on employee's mobile:
  - Mobile Data On
  - GPS / Location On
  - App Background Running

#### Step 3: Video Call Support
- If the issue persists, offer video call/screen share support
- Verify the settings through video conference with client
- Check: WiFi, Settings → Network, etc.

#### Step 4: Permission & App Version Check
You can see the screen and verify:
- Required app permissions are granted
- App is up-to-date with latest version
- No background process restrictions active

### How to Troubleshoot If Location Still Not Showing?

**Common Causes:**
- Admin setting for location tracking is inactive
- Employee mobile setting issue (GPS off, Mobile data off, etc.)
- App permission issue or background restriction
- One employee might not have correct mobile setting, or system-side issue

**Resolution Process:**
1. Check admin panel for location tracking status
2. Verify employee mobile settings (data, GPS, permissions)
3. For persistent issues, offer video call support
4. Escalate to technical support if needed
