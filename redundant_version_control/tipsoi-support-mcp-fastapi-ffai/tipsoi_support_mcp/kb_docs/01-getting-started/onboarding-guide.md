# Tipsoi HRM Platform Onboarding Guide

## What is Tipsoi?

Tipsoi is a comprehensive **HR Management System** (HRM) designed for organizations to manage workforce attendance, leave, payroll, employee records, and reporting. The platform supports:

- **Biometric Attendance Tracking** - Fingerprint, Face Recognition, and RFID devices
- **Mobile Punch** - GPS-based location tracking with mobile app
- **Leave Management** - Policy configuration, applications, and approvals
- **Payroll Processing** - Salary calculations with custom rules
- **Employee Management** - Complete employee lifecycle management
- **Advanced Reporting** - Attendance, leave, late, absent, and overtime reports

## Four Pillars of Tipsoi Success

### 1. HRM/HR Software
The core platform where admins manage all HR operations. Features include dashboard, reports, employee management, and settings.

### 2. Attendance Management System
Track employee attendance using multiple methods:
- Biometric devices (fingerprint, face recognition)
- Mobile app with geofencing
- Web-based punch-in

### 3. Software & Device Communication
Devices communicate with Tipsoi via:
- Internet/Network connection
- WiFi or SIM-based connectivity
- Real-time sync and reporting

## Quick Setup Checklist

When setting up Tipsoi for your organization, follow these steps in order:

### Step 1: Organization Setup
- **Path**: Settings → Organization
- Enter company name, timezone, and industry
- Add office locations and headquarters
- Configure organization hierarchy

### Step 2: Employee Management
- **Path**: Employees → Create Employee
- Add employees individually or bulk upload via Excel
- Assign employee IDs, designations, and departments
- Set up departmental structure and reporting lines

### Step 3: Configure Shift Structure
- **Path**: Shift Management → Create Shift
- Define shift timings (start/end times, break duration)
- Create shift groups for different teams
- Assign shifts to employees or departments

### Step 4: Leave Policy Configuration
- **Path**: Leave → Leave Policy
- Create leave types (Casual, Sick, Annual, etc.)
- Set entitlements per leave type
- Configure accrual methods and carry-forward rules

### Step 5: Set Up Attendance Devices
- **Path**: Devices → Biometric
- Register biometric devices (if using fingerprint/face)
- Configure device communication settings
- Sync devices with employee master data

### Step 6: Configure Payroll (Optional)
- **Path**: Payroll → Settings
- Define salary structures
- Configure overtime rules
- Set up deductions and allowances

## Key Concepts in Tipsoi

### Attendance States
- **Present**: Employee marked as present during shift
- **Absent**: No attendance record for the day
- **Late**: Employee arrived after shift start time
- **On Leave**: Employee has approved leave for the day
- **Holiday**: Company holiday (no attendance expected)
- **Half Day**: Employee present for partial shift

### Leave Management Workflow
1. Employee applies for leave
2. Manager/HR reviews the request
3. Approval/Rejection decision is made
4. System updates leave balance
5. Payroll adjusts salary based on leave

### Payroll Components
- **Salary**: Base monthly compensation
- **Allowances**: Additional payments (house rent, transport, etc.)
- **Deductions**: Taxes, insurance, loans, etc.
- **Overtime**: Extra hours at premium rates
- **Final Pay**: Net salary = Salary + Allowances - Deductions
