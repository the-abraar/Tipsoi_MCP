# Contact Support

## Tipsoi Customer Support Contact

If the chatbot or documentation cannot resolve your issue, contact the Tipsoi Customer Support Team directly:

**Phone:** +8809638017170
**Email:** support@inovacetech.com

Our representatives will respond quickly and provide the necessary assistance.

---

## When to Contact Support

Contact support when:
- Device is not communicating and self-troubleshooting steps have not worked
- Login is completely blocked after verifying credentials
- Data is missing from reports and device sync has been verified
- Employee cannot be created despite following all steps
- Any issue that persists after following the troubleshooting guide

---

## How the Chatbot Escalates to Support

If this chatbot cannot resolve your issue, it will collect the following information and forward to the support team:

- **Customer Name**
- **Company Name**
- **Contact Number**
- **Issue Details**
- **Screenshot or Error Message** (if available)
- **Troubleshooting steps already taken**

After forwarding, the chatbot will inform you:

> "Your issue has been forwarded to our Customer Support Team. Our representative will contact you shortly."

---

## Chatbot Behavior Guidelines

When talking to the chatbot:
- Describe your issue clearly
- Mention which device type you are using (TF 80 / Fast Face / Prompt Series)
- Mention if the issue is for one employee or all employees
- Share any error messages you see on screen

## Frequently Asked Issues (Quick Reference)

| Issue | Quick Action |
|---|---|
| Data not showing / attendance not coming | Check device communication in Admin Panel → Devices |
| Login not working | Re-enter email and password carefully; check internet |
| Location tracking not showing | Enable GPS, mobile data, and app background on employee's phone |
| Face photo not accepted | Use passport-size photo with clear face and solid background |
| Punch shows in log but not in In/Out Time | Punch was outside the allowed Parenting Time window |
| Cannot create employee | Ensure email is unique and password meets requirements |
| Cannot see monthly report | Go to Reports → Attendance Report → select date range |
| Two offices, need to see separate data | Create Workplaces in Settings and assign employees |
| Office time change (e.g., Ramadan) | Create new shift → Edit Roster to assign new shift with effective date |
| Leave not set up | Go to Leave → Create Category → Create Policy → Configure → Assign |
