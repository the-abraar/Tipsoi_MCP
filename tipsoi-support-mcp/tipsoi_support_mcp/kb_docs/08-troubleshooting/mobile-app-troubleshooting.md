# Mobile App Troubleshooting

## Installation & Startup Issues

### Issue 1: App Won't Install

**Solutions**:
1. **Check storage space**: Device needs 100+ MB free
2. **Update OS**: Ensure iOS/Android latest version
3. **Use proper app store**: Download from Google Play / Apple App Store only
4. **Reinstall**: Delete app, restart phone, reinstall

### Issue 2: App Crashes on Startup

**Solutions**:
1. **Force close and restart**: Settings → Apps → Tipsoi → Force Close → Restart
2. **Clear cache**: Settings → Apps → Tipsoi → Storage → Clear Cache
3. **Update app**: Check app store for newer version
4. **Restart phone**: Full phone restart
5. **Uninstall and reinstall**: If above don't work

## Login Issues

### Issue 3: Cannot Login to App

**Solutions**:
1. **Check credentials**: Verify username and password correct
2. **Check internet**: Ensure WiFi/mobile data connected
3. **Reset password**: Use "Forgot Password" on login screen
4. **Check if account disabled**: Contact admin
5. **Update app**: May be compatibility issue with old version

### Issue 4: 2FA Code Not Arriving

**Solutions**:
1. **Check SMS**: Ensure SMS notifications enabled
2. **Wait**: SMS can take 1-2 minutes
3. **Request resend**: Tap "Resend Code"
4. **Check phone number**: Verify registered phone is correct
5. **Use backup method**: Use authenticator app if configured

## Punch-In/Out Issues

### Issue 5: Cannot Punch In

**Problem**: Punch button grayed out or not responding.

**Solutions**:
1. **Check geofence**: If geofencing enabled, ensure at work location
2. **Check time**: Verify current time within shift
3. **Check internet**: Ensure WiFi/data connected
4. **Restart app**: Force close and reopen
5. **Check location services**: Ensure Location permission granted

### Issue 6: Punch Success But Not Recorded

**Solutions**:
1. **Check internet**: Sync may have failed due to poor connection
2. **Force sync**: Pull down to refresh
3. **Try again**: Manual punch might need retry
4. **Contact admin**: If still not showing, admin can manually add

### Issue 7: Geofencing Punch Rejected

**Problem**: "Location outside office geofence" error.

**Solutions**:
1. **Enable location**: Turn on GPS/location services
2. **Try WiFi**: Switch to office WiFi for better location
3. **Move closer**: Ensure within geofence boundary
4. **Restart location**: Disable/enable location services
5. **Manual punch**: If geofence failing, request manager to allow manual

## Data & Sync Issues

### Issue 8: Attendance Not Syncing

**Solutions**:
1. **Check internet**: Use WiFi instead of mobile data
2. **Force sync**: Go to **Profile → Sync**
3. **Restart app**: Force close and reopen
4. **Check pending**: Go to **History** to see pending syncs
5. **Contact support**: If still failing

### Issue 9: Leave Balance Shows Wrong

**Solutions**:
1. **Force refresh**: Pull down on leave balance section
2. **Logout and login**: Refresh login session
3. **Check online**: Verify your internet connected
4. **Update app**: May be showing cached data

### Issue 10: Performance Slow/Laggy

**Solutions**:
1. **Clear cache**: Settings → Apps → Tipsoi → Clear Cache
2. **Reduce apps**: Close other background apps
3. **Free storage**: Delete photos/videos to free up space
4. **Restart phone**: Full restart
5. **Update OS**: Ensure latest iOS/Android version

## Permission Issues

### Issue 11: Cannot Grant Location Permission

**Problem**: Location permission shows as denied.

**iOS Solutions**:
1. Settings → Privacy → Location Services
2. Enable Location Services (toggle ON)
3. Scroll to Tipsoi app
4. Select "Always" or "While Using"

**Android Solutions**:
1. Settings → Apps → Permissions → Location
2. Enable location permission for Tipsoi

### Issue 12: Camera Not Working for ID Verification

**Solutions**:
1. **iOS**: Settings → Privacy → Camera → Enable Tipsoi
2. **Android**: Settings → Apps → Permissions → Camera → Enable Tipsoi
3. **Restart app**: After granting, reopen app
4. **Restart phone**: If still not working

## Notification Issues

### Issue 13: Not Receiving Notifications

**Solutions**:
1. **Check enabled**: Settings → Notifications → Enable Tipsoi notifications
2. **Check sound**: Volume should not be muted
3. **Restart app**: Force close and reopen
4. **Restart phone**: Full restart
5. **Reinstall**: Uninstall and reinstall app

### Issue 14: Too Many Notifications

**Solutions**:
1. Go to **Settings → Notification Preferences**
2. Disable unwanted notification types
3. Adjust frequency (daily digest vs. real-time)
4. Set quiet hours (no notifications during sleep)

## Display Issues

### Issue 15: Text Too Small or Blurry

**Solutions**:
1. **Adjust phone text size**: Phone Settings → Display → Text Size
2. **Update app**: May be rendering bug in old version
3. **Restart app**: Force close and reopen

### Issue 16: App Shows in Different Language

**Solutions**:
1. Go to **Settings → Language**
2. Select your preferred language
3. Restart app to apply

## Data Privacy & Security

### Issue 17: Privacy Concerns

**Important safeguards**:
- App uses HTTPS encryption
- Biometric data never stored on device
- Location data only used for geofencing
- Personal data protected by privacy policy
- Can request data deletion anytime

**To delete your data**:
1. Contact admin
2. Request account deletion
3. All personal data removed from system

## Device Compatibility

### Supported Versions

**iOS**:
- Version: iOS 12.0 or higher
- Devices: iPhone 6s and newer, iPad

**Android**:
- Version: Android 7.0 or higher
- RAM: 2GB minimum (4GB recommended)

**Not supported**:
- Very old phones (> 5 years)
- Tablet-only apps (need phone)
- Rooted/Jailbroken devices (security risk)

### Device Not Supported?

**Solutions**:
1. Use web version: Open browser → Tipsoi login
2. Ask admin for desktop access
3. Upgrade to compatible device
4. Use backup device while yours out of service
